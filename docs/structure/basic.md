# Basic

7大类基础数据类型
#### String
```js
const s = 'abc'
typeof s  // 'string'
```
#### Number
```js
const n = 12
typeof n // 'number'
```
#### Boolean
```js
const b = true
typeof b // 'boolean'
```
#### undefined
```js
const u = undefined
typeof u // "undefined"
```
#### null
```js
const n = null
typeof n  // "object"
```
#### Symbol
```js
const s = Symbol()
typeof s // "symbol"
```
#### bigInt
```js
const b = 12n
typeof b // "bigint"
```
